# Sudoku
Sudoku game using pygame

## Librarys:

- pygame 
- sys
- math

## How To Play 

### Windows - 

- "_pip install pygame_" in cmd.exe
- open folder Memory Game in cmd.exe
- type command "_python main.py_"
- type "_solve_" to solve board or "_reset_" to reset board
